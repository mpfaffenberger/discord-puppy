## Installation

```bash

```

## Configuration

Copy `.env.example` to `.env` and fill in your credentials:

```bash
cp .env.example .env
```

Required settings:
- `DISCORD_TOKEN` - Your Discord bot token
- `ANTHROPIC_API_KEY` - Your Anthropic API key

## Usage

```bash
uv run discord-puppy
```

## Development
